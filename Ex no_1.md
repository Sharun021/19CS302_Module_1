# EX 1 C program to initialize the value as 5.8 & display the same.

## DATE: 08/06/2026

## AIM:

To write a program to initialize the value as 5.8 & display the same.

## Algorithm

1. Start the program.
2. Declare a float variable and initialize it with the value 5.8.
3. Display the value using `printf()`.
4. Stop the program.

## Program:

```c


#include <stdio.h>

int main()
{
    float num = 5.8;

    printf("The value is %.1f", num);

    return 0;
}
```

## Output:

```text
The value is 5.8
```

## Result:

Thus the program was executed and the output was verified successfully.
