# EX 2 C program to check whether the attendance is PRESENT using simple if statement.

## DATE:08/06/2026

## AIM:

To write a program to check whether the attendance is PRESENT using simple if statement.

## Algorithm

1. Start the program.
2. Declare an integer variable for attendance status.
3. Assign the value 1 to indicate PRESENT.
4. Use a simple `if` statement to check whether the value is 1.
5. If true, display "PRESENT".
6. Stop the program.

## Program:

```c
#include <stdio.h>

int main()
{
    int attendance = 1;

    if (attendance == 1)
    {
        printf("PRESENT");
    }

    return 0;
}
```

## Output:

```text
PRESENT
```

## Result:

Thus the program was executed and the output was verified successfully.
